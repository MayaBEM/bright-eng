# Quantifier Mastery

Premium single-file learning web app by Bright EngMath.

## How to use
1. Open `index.html` in a browser.
2. Or deploy the folder to Vercel / Netlify / GitHub Pages.

## Included
- Knowledge sheet
- 3 quiz levels
- 45 questions
- Instant Thai explanations
- Mistake review
- localStorage progress
- Mobile responsive design
